Overview of Fat Burner APIs


Method Endpoint Description
POST /login Login service
GET /api/foods Get Food
GET /api/food Get Food Specifically
GET /api/food/{kategori} Search Food
POST /api/food/log/insert Insert log food
POST /api/food/log/update/{id} Update log food
DEL /api/food/log/delete/{id} Delete log food
GET /api/workouts Get workout
POST /api/workout/log/insert Insert workout
POST /api/workout/log/update/{id} Update workout
DEL /api/workout/log/delete/{id} Delete workout
GET /api/articles Get artikel
POST /api/article/insert Insert artikel
GET /api/workout/log/{id}/{filter} Get Food Report
GET /api/food/log/{id}/{filter} Get Workout Report

API Login


Untuk melakukan autentikasi user


Request


Nama Parameter Tipe Data Keterangan
email String Email akun
password String Password akun

Response


Nama Parameter Tipe Data Keterangan
error Boolean Status error
token String Token
userid String Nomor id user

API Report Workout


Untuk mendapatkan laporan tentang workout yang telah dilakukan


Query string parameters


Nama Parameter Tipe Data Keterangan
id String Nomor id user
filter String Kurun waktu (date,month,year)

Response


Nama Parameter Tipe Data Keterangan
id_log String Nomor id log workout
nama String Nama workout
tanggal String Tanggal workout
waktu_workout String Total waktu workout user
waktu_def String Waktu workout dari sistem
kalori Integer Jumlah kalori
